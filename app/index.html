Modify or replace this file (index.html) to make your "gateway" webpage

This server is running Apache 1.3.4 with PHP 3.0.6 (if that means anything to you) Use .php3 as an extension to have PHP preprocess your web pages.

You can limit access to files with .htaccess files

Everything in your public_html folder will be accessible to the webserver at http://squirrel.bio.brandeis.edu/~yourusername
